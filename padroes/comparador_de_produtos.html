Comparador de produtos

Exemplo do padrão comparador de produtos.

Informações

Os usuários precisam comparar produtos semelhantes.

Em sites de compras, produtos da empresa, sites de leilão, os usuários chegaram a uma página do produto ou chegaram a uma lista de produtos. Agora eles precisam decidir pelo produto ou produtos similares. Os usuários podem diferir no que consideram importante para um produto. Os usuários devem poder comparar facilmente os recursos individuais do produto.

Mostre os recursos nas linhas da matriz e os produtos nas colunas. Se um recurso estiver presente, marque-o. Caso contrário, não marque. Se o recurso tiver dados interessantes específicos para os usuários, mostre os dados em vez da marca.

Se houver muitos recursos, eles podem ser agrupados para que os usuários possam selecionar quais aspectos eles desejam comparar. Quando houver muitos produtos para comparar, permita que os usuários selecionem quais produtos eles desejam comparar. Isso pode ser feito em uma página quando não há muitos produtos, caso contrário, é feito em duas etapas.

A matriz de produtos/recursos fornece uma visão geral imediata e facilita a comparação. Marcando apenas os recursos disponíveis, a desordem visual é minimizada.
