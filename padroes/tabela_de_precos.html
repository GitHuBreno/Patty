Tabela de preços

Exemplo do padrão tabela de preços.

Informações

O usuário precisa ter uma visão geral de quais planos de preços são oferecidos e como eles diferem.

  • Use quando houver mais de uma versão de um produto para escolher entre.
  • Use quando desejar fornecer ao usuário uma visão geral de como as versões dos produtos diferem em termos de recursos e preço.
  • Não use quando o usuário puder comprar mais de um produto por vez.
  • Não use quando o usuário puder decidir quais produtos ele deseja comprar em uma sessão e retornar mais tarde para realizar o pagamento real.

As tabelas de preços são usadas como uma maneira de ilustrar como os recursos de um produto diferem conforme o preço muda.

Exiba a versão diferente de um produto em uma tabela, alinhando preço e recursos para comparação.

Uma lista das perguntas mais frequentes (FAQ) sobre o produto é frequentemente listada diretamente abaixo da tabela de preços. Elas geralmente tratam de questões com as quais os clientes em potencial normalmente se preocupam: como funciona o plano gratuito, existe uma garantia de devolução do dinheiro, como serei cobrado etc.

É uma parte muito comum dos sites de marketing de Application Service Providers (ASPs). Na maioria dos casos, esses produtos possuem apenas um produto importante, mas oferecem esse produto em diferentes variantes (planos). Nesses sites, o preço geralmente é baseado em um plano de assinatura mensal/trimestral/anual.

Ao criar sua tabela de preços, é bom considerar os seguintes pontos:

  • Separe visualmente os planos usando cores de fundo alternadas. Quando usado com moderação, você pode atrair a atenção do plano que deseja que o usuário compre.
  • Utilize diferentes tamanhos e cores de fontes para os elementos que você deseja destacar: títulos, preços, títulos, etc.
  • Esteja ciente de que os usuários rolam para baixo em tabelas longas. Os preços na parte superior de uma tabela podem não estar visíveis quando atingem o fundo. Uma solução é colocar os preços na parte superior e inferior ou outra é manter a tabela de preços curta.

Converter visitantes interessados ​​em clientes pagantes é o seu maior objetivo. Use tabelas de preços para ilustrar do que o seu produto é capaz de fornecer e ao mesmo tempo atraí-lo.
